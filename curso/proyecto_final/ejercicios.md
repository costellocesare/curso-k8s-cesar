# Ejercicios prácticos del Proyecto Final

Pon a prueba tus conocimientos resolviendo estos ejercicios:

1. Modifica el backend para agregar una nueva funcionalidad (por ejemplo, eliminar notas).
2. Cambia el número de réplicas del backend en el manifiesto Deployment y observa el comportamiento.
3. Sube tu imagen Docker personalizada a Docker Hub y actualiza el manifiesto para usarla.
4. Implementa un ConfigMap para cambiar el título del frontend sin modificar el código fuente.
5. Simula una caída de la base de datos y observa la resiliencia del clúster.
6. Agrega una Readiness Probe personalizada al backend.
7. Realiza un Rolling Update del backend cambiando la versión de la imagen.

¡Comparte tus resultados y dudas en la comunidad del curso!
